# GPSQFramework

هذا هو الفريم المنظم لمشروع gpsq.

## الأقسام

- `Headers/` التعريفات والإعدادات.
- `Core/` التفعيل، UUID، الشبكة، التخزين.
- `UI/` الواجهة والمكونات.
- `Map/` الخريطة والبحث والمفضلة.
- `Utils/` أدوات مساعدة.
- `Resources/` ملفات اللغة والصور لاحقًا.

> ملاحظة: هذا فريم تنظيمي داخل مشروع Theos، وليس Framework ديناميكي مستقل بصيغة `.framework`.
