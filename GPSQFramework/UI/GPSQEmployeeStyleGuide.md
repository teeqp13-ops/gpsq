# GPSQ EmployeeSwitch Inspired Style Guide

تم استلهام هذا التنظيم من حزمة EmployeeSwitch.App بدون نسخ مكتبات Xamarin أو ملفات DLL.

## عناصر مستفادة

- واجهة منظمة بمكونات منفصلة.
- دعم موارد ولغات.
- طبقة خرائط.
- طبقة هوية وتفعيل.
- طبقة تخزين آمنة لاحقًا.
- قابلية إضافة صور وأيقونات وخطوط.

## ملاحظات مهمة

- لا يتم دمج ملفات `.dll` أو ملفات Xamarin مباشرة داخل Theos.
- يتم إعادة بناء الأفكار على Objective-C++ فقط.
- ملفات `GPSQFramework` هدفها تنظيم المشروع وتسهيل التطوير.
